The calculateHash() method calculates the hash value of the block.

It starts by concatenating the block's data, previousHash, and timestamp into a single string called data.

It then creates a new Uint8Array of length 32 to hold the hash value.

The data string is encoded into bytes using TextEncoder().encode(data).

A loop iterates from 0 to 31 (32 iterations) and assigns each byte from dataAsBytes to the corresponding index in the hash array using the modulo operator to wrap around the bytes if necessary.

Finally, the calculated hash is returned as a string.

    calculateHash() {
    const data = this.data + this.previousHash + this.timestamp;
    let hash = new Uint8Array(32);
    const dataAsBytes = new TextEncoder().encode(data);
    for (let i = 0; i < 32; i++) {
        hash[i] = dataAsBytes[i % dataAsBytes.length];
    }
    return hash.toString();
}

Python

The hashlib module provides various hashing algorithms, such as SHA-256, which we'll use to calculate hash values.


The datetime module is used to work with dates and times.


import hashlib
import datetime

This defines the Block class, which represents a single block in the blockchain.

The __init__ method is the constructor that initializes the block's properties: timestamp, data, previousHash, and hash.

self.timestamp stores the current timestamp using datetime.datetime.now().

self.data stores the data associated with the block.

self.previousHash stores the hash value of the previous block.

self.hash is initialized by calling the calculateHash() method, which generates the hash value for the current block based on its properties.

The calculateHash() method concatenates the block's data, previous hash, and timestamp into a single string. It then encodes the string, calculates the SHA-256 hash using hashlib.sha256(), and returns the hexadecimal representation of the hash using hexdigest().


class Block:
    def __init__(self, data, previousHash):
        self.timestamp = datetime.datetime.now()
        self.data = data
        self.previousHash = previousHash
        self.hash = self.calculateHash()

    def calculateHash(self):
        data = str(self.data) + str(self.previousHash) + str(self.timestamp)
        return hashlib.sha256(data.encode()).hexdigest()

This defines the Blockchain class, which manages the chain of blocks.

The __init__ method initializes the chain with a single block, the genesis block, created by calling the createGenesisBlock() method.

The createGenesisBlock() method creates and returns the genesis block by passing "Genesis Block" as the data and "0" as the previous hash to the Block constructor.

The addBlock() method adds a new block to the chain. It retrieves the previous block from the chain, creates a new block by passing the data and previous hash to the Block constructor, and appends it to the chain.

The isValid() method checks the validity of the blockchain. It iterates through the chain starting from the second block and verifies that the hash of each block matches the calculated hash and that the previous hash of each block matches the hash of the previous block. If any inconsistency is found, it returns False. If the loop completes without any issues, it returns True.


Full source code

class Blockchain:
    def __init__(self):
        self.chain = [self.createGenesisBlock()]

    def createGenesisBlock(self):
        return Block("Genesis Block", "0")

    def addBlock(self, data):
        previousBlock = self.chain[-1]
        newBlock = Block(data, previousBlock.hash)
        self.chain.append(newBlock)

    def isValid(self):
        for i in range(1, len(self.chain)):
            if self.chain[i].hash != self.chain[i].calculateHash():
                return False
            if self.chain[i - 1].hash != self.chain[i].previousHash:
                return False
        return True

C language

These are the header files included for standard input/output operations (stdio.h), time-related functions (time.h), string manipulation functions (string.h), and dynamic memory allocation functions (stdlib.h).


#include <stdio.h>
#include <time.h>
#include <string.h>
#include <stdlib.h>

This defines a structure called Block to represent a single block in the blockchain.

It contains members such as timestamp (of type time_t to store the timestamp), data (a pointer to store the block's data), previousHash (a pointer to store the hash of the previous block), and hash (a pointer to store the hash of the current block).


typedef struct Block {
    time_t timestamp;
    char *data;
    char *previousHash;
    char *hash;
} Block;

This function calculates the hash value for a given block.

It concatenates the block's data, previous hash, and timestamp into a single string using sprintf.

Then, it dynamically allocates memory for the hash using malloc to store the hash value.

The hash value is calculated by copying characters from the concatenated string in a cyclic manner.

Finally, the calculated hash is returned.


char *calculateHash(Block *block) {
    char data[256];
    sprintf(data, "%s%s%ld", block->data, block->previousHash, (long)block->timestamp);
    char *hash = (char*)malloc(sizeof(char) * 32);
    for (int i = 0; i < 32; i++) {
        hash[i] = data[i % strlen(data)];
    }
    return hash;
}


This function creates the genesis block, which is the first block in the blockchain.

It allocates memory for a new Block structure, assigns the current timestamp using time(NULL), sets the data as "Genesis Block", assigns "0" as the previous hash, and calculates the hash value using the calculateHash function.

Finally, it returns the genesis block.


Block *createGenesisBlock() {
    Block *genesisBlock = (Block *)malloc(sizeof(Block));
    genesisBlock->timestamp = time(NULL);
    genesisBlock->data = "Genesis Block";
    genesisBlock->previousHash = "0";
    genesisBlock->hash = calculateHash(genesisBlock);
    return genesisBlock;
}


This defines a structure called Blockchain to represent the entire blockchain.

It contains members such as chain (a pointer to an array of Block pointers) and size (to store the size of the blockchain).


typedef struct Blockchain {
    Block **chain;
    int size;
} Blockchain;


This function creates a new blockchain.

It allocates memory for a new Blockchain structure, assigns memory for the chain array with a size of one Block pointer, sets the first element of chain to the genesis block created by calling createGenesisBlock, and initializes the size of the blockchain to 1.

Finally, it returns the newly created blockchain.


Blockchain *createBlockchain() {
    Blockchain *blockchain = (Blockchain *)malloc(sizeof(Blockchain));
    blockchain->chain = (Block **)malloc(sizeof(Block *));
    blockchain->chain[0] = createGenesisBlock();
    blockchain->size = 1;
    return blockchain;
}


This function adds a new block to the blockchain.

It takes the blockchain as input, along with the data for the new block.

It retrieves the previous block from the chain using the size information, allocates memory for a new Block structure, sets the timestamp to the current time, assigns the data, sets the previous hash as the hash of the previous block, calculates the hash for the new block using the calculateHash function, increments the size of the blockchain, reallocates memory for the chain array to accommodate the new block, and finally, adds the new block to the chain.


void addBlock(Blockchain *blockchain, char *data) {
    Block *previousBlock = blockchain->chain[blockchain->size - 1];
    Block *newBlock = (Block *)malloc(sizeof(Block));
    newBlock->timestamp = time(NULL);
    newBlock->data = data;
    newBlock->previousHash = previousBlock->hash;
    newBlock->hash = calculateHash(newBlock);
    blockchain->size++;
    blockchain->chain = (Block **)realloc(blockchain->chain, sizeof(Block *) * blockchain->size);
    blockchain->chain[blockchain->size - 1] = newBlock;
}


This function checks the validity of the blockchain.

It iterates through the chain, starting from the second block.

For each block, it compares the stored hash with the calculated hash using strcmp to check if they match. If not, it returns 0 to indicate the blockchain is invalid.

It also compares the previous hash of the current block with the hash of the previous block to ensure the chain is intact. If they don't match, it returns 0.

If the loop completes without any issues, it returns 1, indicating that the blockchain is valid.


int isValid(Blockchain *blockchain) {
    for (int i = 1; i < blockchain->size; i++) {
        if (strcmp(blockchain->chain[i]->hash, calculateHash(blockchain->chain[i])) != 0) {
            return 0;
        }
        if (strcmp(blockchain->chain[i - 1]->hash, blockchain->chain[i]->previousHash) != 0) {
            return 0;
        }
    }
    return 1;
}


The main function is the entry point of the program.

It creates a new blockchain by calling createBlockchain, adds two blocks to the blockchain using addBlock with different transaction data, checks the validity of the blockchain using isValid, and prints the result to the console using printf.


int main() {
    Blockchain *blockchain = createBlockchain();
    addBlock(blockchain, "Transaction 1");
    addBlock(blockchain, "Transaction 2");
    printf("Blockchain is valid: %d\n", isValid(blockchain));
    return 0;
}
